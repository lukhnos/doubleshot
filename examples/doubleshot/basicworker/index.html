DoubleShot BasicWorker Example

Synopsis

This is a very simple app that shows you how to use the concurrency primitives provided by the DoubleShot. DoubleShot is a CoffeeScript dialect that adds cross-platform concurrency support to the base language. Read more about it on github.

For the source code and a quick code walk-through, please refer to this page for details. It also provides the instructions on how to run this app as a node.js program.
